# Neural Network Training
Exercise in training a neural network using time-series data to predict the price of goods in the future.

Using PyTorch to train the neural network (gradient descent, learning rate, backpropagation, neural network layers, ReLU, creating the tensors, defining the neural network model).

Using NumPy for mathematical operations such as generating the sample data and normalizing the data.

Using Pandas to read the CSV and convert the data from the database to usable objects.
